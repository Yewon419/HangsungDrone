# 실내 특화 드론 라이트쇼 사업 — 회의 정리 v0.2

> 작성일: 2026-05-06
> 단계: 정체성 락 완료 (Pre-MVP)
> 상태: 정체성 5개·L1.5 데뷔·L3 자체 BOM·시스템 아키텍처 락. 설계 4개 진입 대기

---

## TL;DR

**한국 e모빌리티·이차전지 산업 박람회를 첫 ICP로 한 실내 특화 저가 드론쇼.** 글로벌·국내 누구도 만들지 않은 *30-50대 단위 박람회 부스·소형 메인 시연* 카테고리를 신설. Crazyflie 2.1+ 기반 Lighthouse Positioning + Skybrush 오픈소스 스택. 데뷔 = L1.5 (Crazyflie 30대 + 풀 파이프라인 + 반자동 SaaS), 12-18개월 후 시연 진입.

## 1. 사업 정체성 (락 완료)

| # | 항목 | 답 |
|---|---|---|
| WHO | 첫 핵심 고객 | e모빌리티·이차전지 산업 박람회 (주최측 + 참가 기업), 실내 전용 |
| WHAT | 제품 본질 | 저가 카테고리(시장 신설)의 산업 컨텍스트 시각화 융합형 드론쇼 |
| JTBD | 고객이 해결하려는 일 | 박람회 PR 자산 + 부스 임팩트 + 고가 메가쇼 못 부르는 부스·소형 메인 시장 채움 |
| MOAT | 경쟁 우위 | 30-50대 단위 가격 효율화 + 산업 도메인 이해 + 융합형 + 운영 안전 패턴 + 반자동 SaaS 즉시 견적·미리보기 경험 |
| MODEL | 수익 구조 | 박람회·부스 단위 + 주기성(시즌제) + 객단가 200-500만 추정 (글로벌 옥외 100대급 1/5) |

**저가 정의 = 시장 신설형 (i 아닌 ii)**: 절대 단가 압박이 아니라 *시장에 없던 가격대 자체를 새로 만듦*. 30-50대 단위 = 어느 기존 업체도 다루지 않는 단위. BOM은 표준 안전 등급 유지하면서도 *시장 단위*가 작아 객단가가 낮아짐.

## 2. 시장 분석

### 2.1 글로벌 단가 기준 (옥외 쇼, 참고용)

- 드론당 $200~$300, 100대 최소 → base $20K~$30K
- 중간 규모 300대 $60K~$90K, 미국 평균 쇼당 $50K
- B2C 최저가: 100대 패키지 $17K~$25K

### 2.2 한국 3대 업체 (참고용 — 우리는 다른 시장)

| 항목 | 다온아이앤씨 | 유비파이 (UVify) | 파블로항공 |
|---|---|---|---|
| 비행제어 | 자체 개발 | PX4 오픈소스 + 자체 보강 | 자체 개발 |
| 디자인 SW | 자체 콘텐츠팀 | 자체 통합 SW | NeoPicasso (AI 기반) |
| 운영 SW | 자체 | 자체 | PADSS |
| 위치측위 | RTK-GNSS | RTK + PX4 EKF | RTK + 분산 통신 |
| 최대 운용 | 1,050대 (광안리) | 1만대 단일 + 5만2천대 기네스 4관왕 | 1,068대 LED+불꽃 |
| 차별 포지션 | 플렉서블 레이아웃, 국방 | 글로벌 SW 표준 호환 | 콘텐츠 다각화(불꽃/수상) |

→ **3사 모두 옥외·대형 메가쇼 위주.** 실내 박람회 30-50대 단위는 누구도 안 함.

### 2.3 글로벌 벤치마크: Verity Studios (스위스)

- **Lucie® 마이크로 드론**: 50g 미만, 매우 조용함
- **Stage Flyer**: Cirque du Soleil "Paramour" 무대용
- 레퍼런스: TED 2016 (33대) → Eurovision 2025 (104대) → Royal Caribbean 크루즈 상설 (48대, 180회+ 무사고)
- ETH Zurich D'Andrea 교수의 13년+ R&D
- **진입장벽: Failsafe 기술** (단일 부품 실패에도 통제된 비행)

→ Verity는 대형 엔터테인먼트 집중, 산업 박람회 시장은 안 함. **참고만, 우리는 안전 IP 영역 안 따라감 (운영 패턴으로 보강).**

### 2.4 우리 시장: e모빌리티 박람회 = 빈 카테고리

- 다온/유비파이/파블로 = 옥외 메가쇼 (최소 100대)
- Verity = 대형 엔터테인먼트 (해외)
- **30-50대 단위 박람회 부스·소형 메인 시연 = 글로벌·국내 모두 비어있는 카테고리**
- 산업 컨텍스트 시각화(전기차·배터리·충전망 시각화) = 어디도 안 함

### 3.1 핵심 결정 (락)

- **포지셔닝**: Lighthouse 주 (드론 자체 위치 인식) + 카메라 보조 (안전 백업·이상 탐지)
- **명령 구조**: 사전 안무 다운로드 + 시작·sync·비상정지·미세 보정 하이브리드 (continuous command 아님)
- **자율 안전 동작**: 펌웨어 내장 필수 — 외부 카메라 reactive 보정은 latency 50-120ms로 큰 이탈 보정 불가능, 자체 안전 동작만이 신뢰 가능

## 4. 기술 스택

### 4.1 L1.5 데뷔 사양 (Crazyflie 기반)

| 항목 | 사양 |
|---|---|
| 모델 | Crazyflie 2.1+ (Bitcraze, 스웨덴) |
| MCU | STM32F405 (168MHz, Cortex-M4F) |
| IMU | BMI088 (6축) |
| 자력계 | BMM150 |
| 위치 인식 | Lighthouse Positioning Deck (별매, 필수) |
| LED | 기본 1개 (약함) → LED Ring Deck 또는 자체 LED 보드 추가 |
| 배터리 | LiPo 1S 250mAh, 비행시간 5-7분 |
| 무게 | 27g (베어), ~35g (덱 포함) |
| 라디오 | 2.4GHz nRF51822, Crazyradio 2.0 동글 |
| 펌웨어 | 오픈소스, 자율 안전 일부 내장 (저배터리 자동 착륙, 통신 두절 호버) |

**견적 (한국 수입 추정, Phase 0 정식 검증 필요):**
- 드론 30대 (덱 포함) ≈ 1,380만
- Lighthouse Base V2 ×4 ≈ 150만
- Crazyradio 2.0 ×2 ≈ 15만
- **하드웨어 총 ≈ 1,545만**

직수입(Bitcraze 본사 또는 DigiKey/Mouser)은 30-40% 저렴 가능.

### 4.2 Lighthouse Positioning 사양

- IR 베이스스테이션이 빔 송출, 드론 광검출기가 위치 계산
- 5×5m 공간 mm 단위 상대정밀도, cm 단위 절대정밀도
- 모션캡처(VICON) 대비 압도적 저렴
- **한계**: 베이스 V2 사거리 6m, 4개로 5×5×2m 커버. 그 이상은 측면 추가

### 4.3 L3 자체 BOM ((b) 표준 등급)

| 부품 | 후보 | 단가 |
|---|---|---|
| MCU | ESP32-S3 / RP2040 | 5천원 이하 |
| 모터+프롭 ×4 | 7mm 코어리스 | 5천원 |
| LiPo 1S | 150-300mAh | 3천원 |
| RGB LED | 고휘도 5050 | 1천원 |
| IMU | MPU-6500 / ICM-42688 | 3천원 |
| Lighthouse 광검출기 | TS4231 등 | 5천원 |
| 프레임 | 3D 프린팅 PLA | 1천원 |

**대당 BOM ≈ 2-4만원** (Crazyflie 25-30만원 대비 1/10)

- Verity Failsafe IP 영역 안 따라감
- 펌웨어·통신 프로토콜은 직접 작성 (자율 호버, 자동 착륙, 통신 두절 대응)
- 100대+ 확장 시 단가 효율 결정적

## 5. 디자인 시스템

### 5.1 (b1) 패턴 라이브러리 + 알고리즘 보간

- 안무 패턴 라이브러리 30-50개 (formation 변환, 시간 기반 효과)
- 알고리즘 자동 보간 (패턴 간 트랜지션, 드론 수 가변, 색·시간 파라미터)
- 충돌·물리 한계 자동 검증 (속도·가속도·배터리)
- **오픈소스 활용 우선**: Skybrush 자체, Crazyswarm2 (ROS2+Crazyflie), formation transition 라이브러리, 학계 안무 생성 코드 등 → Phase 0 리서치 필요
- 진짜 자동 생성(자연어/이미지 입력) (b2)는 학계 R&D 영역 → 안 함

### 5.2 3D 시뮬레이션 미리보기

- Skybrush Live 활용 (가상 드론 안무 실행)
- 클라이언트 프레젠테이션용 핵심 자산
- 실내 환경 = 바람·외란 무시 → 운동학 시뮬레이션만으로 충분

### 5.3 사람 검수/조정 (반자동)

- 알고리즘 자동 생성 → 사람(사용자 또는 디자이너)이 검수·미세 조정
- 사업 확장 시 전문 안무 디자이너 고용 옵션

## 6. 주문·관리 시스템 (iv) 반자동 SaaS

| 단계 | 자동/수동 |
|---|---|
| 캘린더 행사일·시간 예약 | 자동 |
| 정형 폼 작성 (행사 정보) | 자동 (사용자) |
| 사용자(오너)에게 전달 | 자동 |
| 견적 산출 | 수동 (오너) |
| 견적 OK | 수동 (고객) |
| 결제 진행 | 자동 |
| 일정 관리 | 자동 |
| 디자인 미리보기 (3D) | 자동 |
| 계약서 자동 생성 | 자동 |

박람회 커스텀 견적 시장에는 자동 견적이 부적합 → 반자동이 합리적. 자동 견적은 표준 SKU 시장(웨딩 패키지 등) 이야기.

## 7. 안전 모델

### 7.1 운영 안전 패턴 (1차 방어선)
- 저고도 운용
- 객석·동선 분리 (드론 공역과 관객 공역 물리적 격리)
- E-Stop 물리 버튼 (사람이 누르는 비상정지)
- Geofence (드론이 절대 못 가는 영역 정의)

### 7.2 펌웨어 자율 안전 (2차 방어선)
- 저배터리 자동 착륙
- 통신 두절 N초 시 자동 호버 → 착륙
- IMU 이상 신호 감지 시 자체 호버

### 7.3 안전 대안 (운영 환경별 선택)

1. **Tethered (테더드)**: 천장 케블라 줄로 추락 방지, 부스 환경에 적합
2. 풀 케이지 / 덕트 프롭 (Tello-EDU 방식)
3. 저고도 + 캐치넷 (객석 위 비행 금지)
4. 소프트 바디 (20g 이하 + 부드러운 프레임)

→ 박람회 부스 시연이면 (3) 저고도+객석분리가 기본, 천장 트러스 가능 환경이면 (1) Tethered 추가 가능.

## 8. 한국 규제

### 8.1 옥외 (참고용 — 우리 시장 아님)
- 항공안전법 제129조 5항: 야간비행/비가시권은 특별비행승인 필수
- 지방항공청장 30~90일 검사 후 승인

### 8.2 실내 = 규제 우회 경로
- 항공안전법은 옥외 공역 기준 → **실내 군집비행은 거의 규제 없음**
- 사업자 등록 필요, 야간비행 특별승인 불필요
